TH: A fork of another persons pdf_downloader script, that i attemptet to both improve and test

Original README below:

----------


Python code to be used to download .pdf files from a specific column in an Excel sheet.


Code can be customized to change the paths for the Excel file and output for downloads.
Will also require an "ID" column to name the files.
Outputs logs into the output folder


Other customizeable options include:

Amount of workers (since this program uses multi-threading) (default = 3)

Amount of files downloaded (default = 10)

Amount of time before a timeout (default = 60 seconds)
